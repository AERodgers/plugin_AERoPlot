plugin_formantPlotter v.1.0.0

A plugin to help to draw pretty formant plots in Praat .

  1. Click on the clone or download button above followed by download ZIP.
  2. Download and unzip the file.
  3. Copy the plugin_phonetics_practice folder and all its contents into the Praat preferences directory.

(Click on this link for guidelines on how to locate this directory on your computer.)

Once you have installed the plugin, you can access it from New > graphic > Formant Plot the objects window.
